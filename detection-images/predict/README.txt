==================模型预测说明====================

一、环境安装
1. 安装CUDA 10.0，用于GPU预测
3. 使用 
	pip install -r requirements.txt
    安装训练所需的库

==============================================

二、使用说明
1. 将训练得到的checkpoint.pth放在本文件夹下
2. 离线预测
    （1）将_arr.txt结尾的文件放在in文件夹下，可以一次放入多个文件
    （2）运行 python predict.py，进行预测
    （3）模型输出结果放在out文件夹下，和输入文件同名，以_label.txt结尾
3. 在线预测
    （1）运行
	python predict.py --online
             将监听socket 127.0.0.1:28889
    （2）通过程序向socket传输数据，数据格式为
	长度 + \n + 原输入的_arr.txt中相同格式的内容
    （3）模型预测完毕后将通过socket向程序返回数据，数据格式为
	长度 + \n + 原预测的_label.txt中相同格式的内容

==============================================

三、参数说明
 -i	--input-dir	离线预测时，输入文件的位置，默认in
 -o	--output-dir	离线预测时，输出文件的位置，默认out
	--online		开启在线预测
	--port		在线预测时，模型监听的端口号，默认28889

 -b	--batch-size	模型一次处理的切分后的数据的数量，
			值越大，内存占用越高，处理速度越快
			默认值为8
 -n	--num-workers	数据加载时使用的子进程的数量，
			值越大，内存占用越高，处理速度越快
			默认值为8
	--box-score-thresh	输出边界框的阈值，值越大，输出的边界框越少，
			默认值为0.05
	--box-nms-thresh	边界框做非最大抑制时的阈值，值越大，输出的
			边界框越多，默认值为0.5
